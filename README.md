# Depot application

## Getting started
Clone the repository
```
$ git clone https://github.com/michudzik/depot <destination>
```

Install required gems
```
$ bundle install
```

Migrate the database
```
$ rails db:migrate
```

Run the test suite
```
$ rails test
```

Start the server
```
$ rails server
```
